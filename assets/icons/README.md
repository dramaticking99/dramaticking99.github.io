# Icons Directory

This directory contains icons and favicon files for your portfolio website.

## Required Files:

### favicon.ico
- Website favicon (browser tab icon)
- Recommended sizes: 16x16, 32x32, 48x48
- ICO format for maximum compatibility

## How to Create a Favicon:

### Method 1: Online Favicon Generator
1. Use a service like favicon.io, realfavicongenerator.net, or favicon-generator.org
2. Upload your logo or create text-based favicon
3. Download the generated ICO file
4. Rename to `favicon.ico` and place in this directory

### Method 2: Design Tools
1. Create a 32x32px or 64x64px square image
2. Use simple, recognizable design (initials, logo, symbol)
3. Export as ICO file or use online converter
4. Name it `favicon.ico`

## Tips for Good Favicon Design:
- Keep it simple and recognizable at small sizes
- Use high contrast colors
- Consider your brand colors
- Test at different sizes (16px, 32px)
- Make sure it's readable on both light and dark backgrounds

The favicon is already linked in the HTML head section and will automatically appear once you place the file here.
